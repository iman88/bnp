# BridgeNewProficiency
*Punte spre noi competențe*

Trasee de învățare structurate pentru cei care lucrează sau vor să lucreze în testare software — indiferent dacă ești la început, vrei să te actualizezi sau să avansezi în carieră. Fără costuri, fără cursuri plătite.

Resursele sunt selectate din platforme recunoscute la nivel internațional: Test Automation University, Playwright Docs, freeCodeCamp, Hyperskill, Microsoft Learn. Fiecare traseu e organizat ca un plan clar: ce înveți, în ce ordine, cât durează și pentru ce rol ești pregătit la final.

> **Notă privind limba:** Toate resursele sunt în limba engleză. Sunt necesare cunoștințe medii pentru a le parcurge fără dificultate.

## 📂 Toate traseele

### 1️⃣ Fundamente — baza oricărui QA

> Indiferent de direcția pe care o alegi mai târziu, acestea sunt competențele de bază.

| Traseu | Focus | Durată |
|--------|-------|--------|
| [Testare Manuală](./cale-invatare-testare-manuala.md) | Fundamente QA · Bug reporting · Unelte · ISTQB | ~3–5 luni |
| [Testare API cu Postman](./cale-invatare-postman.md) | API testing · Colecții · Automatizare Postman | ~8–12h |
| [Testare SQL & Baze de Date](./cale-invatare-sql-testing.md) | SQL · Validare date · Integritate BD · Automatizare | ~2–3 luni |

### 2️⃣ Automatizare — alege UN singur traseu

> Nu trebuie să le înveți pe toate. Alege unul și du-l până la capăt — toate duc la același nivel de angajabilitate.

| Traseu | Limbaj & Framework | Durată |
|--------|--------------------|--------|
| ⭐ [TypeScript + Playwright](./cale-invatare-de-la-testare-manuala-la-automatizare-typescript.md) | TypeScript · Playwright | ~4–6 luni |
| [Python + Playwright](./cale-invatare-de-la-testare-manuala-la-automatizare-python.md) | Python · Playwright | ~4–6 luni |
| [Python + Selenium](./cale-invatare-de-la-testare-manuala-la-automatizare-python-selenium.md) | Python · Selenium WebDriver | ~4–6 luni |
| [Java + Playwright](./cale-invatare-de-la-testare-manuala-la-automatizare-java-playwright.md) | Java · Playwright | ~5–7 luni |
| [Java + Selenium](./cale-invatare-de-la-testare-manuala-la-automatizare-java.md) | Java · Selenium · REST Assured | ~5–7 luni |
| [.NET + Playwright](./cale-invatare-de-la-testare-manuala-la-automatizare-dotnet.md) | C# · .NET · Playwright | ~5–7 luni |

**⭐ Recomandare pentru 2026–2027:** TypeScript + Playwright — cea mai cerută combinație în anunțurile de QA Automation, documentație excelentă și ecosistem web nativ.

> **Cum alegi limbajul:**
> - Pornești de la zero → **TypeScript + Playwright** ⭐
> - Vrei sintaxa cea mai simplă sau combini cu AI/data science → **Python + Playwright**
> - Echipa ta lucrează cu Selenium → **Python + Selenium** sau **Java + Selenium**
> - Lucrezi în enterprise sau ecosistem Microsoft → **Java** sau **.NET**

### 3️⃣ Specializări tehnice

> Pot fi abordate după ce ai fundamente solide. Testarea manuală mobilă — oricând; restul — după 1–2 ani de experiență.

| Traseu | Focus | Durată |
|--------|-------|--------|
| [Mobile Testing](./cale-invatare-mobile-testing.md) | Android · iOS · Appium · Testare în cloud | ~4–5 luni |
| [Security Testing](./cale-invatare-security-testing.md) | OWASP Top 10 · Burp Suite · API Security | ~3–5 luni |
| [Performance Testing](./cale-invatare-performance-testing.md) | JMeter · k6 · Analiză blocaje · CI/CD | ~3–4 luni |
| [SDET](./cale-invatare-sdet.md) | Infrastructură testare · Cod calitate producție · Observabilitate | ~6–9 luni |

### 4️⃣ Roluri de avansare

> Necesită experiență solidă de minimum 3–5 ani. Nu sunt trasee tehnice — sunt trasee de carieră.

| Traseu | Focus | Nivel de start | Durată |
|--------|-------|---------------|--------|
| [QA Tech Lead](./cale-invatare-qa-tech-lead.md) | Mentorat · Procese QA · Reprezentare tehnică | Senior QA | ~4–5 luni |
| [QA Release Manager](./cale-invatare-qa-release-manager.md) | Procese lansare · Coordonare echipe · Metrici DORA | Senior QA | ~4–6 luni |
| [Test Arhitect](./cale-invatare-test-arhitect.md) | Strategie testare · Arhitectură framework · Leadership tehnic | Senior QA | ~5–7 luni |
| [QA Manager](./cale-invatare-qa-manager.md) | Management echipă · Hiring · Buget · Strategie | QA Tech Lead | ~5–7 luni |

**Cum alegi între Tech Lead, Release Manager și Test Arhitect:**

| | QA Tech Lead | QA Release Manager | Test Arhitect |
|-|-------------|-------------------|---------------|
| **Focus** | Oamenii din echipa de QA | Procesul de lansare | Arhitectura și strategia de testare |
| **Lucrezi cu** | Echipa de QA + dev | DevOps · PO · Management | Arhitecți · CTO · toți tech lead-ii |
| **Orizont** | Sprint, lună | Lansare, trimestru | 1–2 ani |
| **Dacă îți place să...** | Crești oameni | Coordonezi procese | Gândești sistemic pe termen lung |

### 5️⃣ AI în Testare

> Poate fi adăugat oricând, indiferent de nivelul tehnic. Alege varianta potrivită rolului tău.

| Traseu | Pentru cine | Focus | Durată |
|--------|-------------|-------|--------|
| [AI pentru Testeri Manuali](./cale-invatare-ai-in-testare-manual.md) | Testeri manuali, fără cod | Prompt engineering · Platforme no-code · Analiză buguri | ~1–2 luni |
| [AI pentru Automation Engineers](./cale-invatare-ai-in-testare-automatizare.md) | Automation QA cu experiență | Copilot · Cursor · Generare teste · Autonomous testing | ~2–3 luni |
| [AI în Testare — Full Stack](./cale-invatare-ai-in-testare.md) | QA Tech Lead · Test Arhitect · Senior cu 5+ ani experiență | Tot ce e mai sus + CI/CD cu AI · Strategie la nivel de echipă | ~3–4 luni |

> **Cum alegi:** Dacă nu scrii cod → **AI pentru Testeri Manuali**. Dacă scrii teste automate → **AI pentru Automation Engineers**. Full Stack e pentru cei care conduc echipe sau definesc strategia de testare la nivel de organizație — nu e un pas obligatoriu, e o direcție separată.
